# Todos Feature (Example)

This is a reference implementation. Copy this pattern for your feature.

## Files

- `routes.ts` - OpenAPI route definitions
- `service.ts` - Business logic, database queries

## Endpoints

| Method | Path           | Description |
| ------ | -------------- | ----------- |
| GET    | /api/todos     | List all    |
| GET    | /api/todos/:id | Get by ID   |
| POST   | /api/todos     | Create      |
| PATCH  | /api/todos/:id | Update      |
| DELETE | /api/todos/:id | Delete      |
